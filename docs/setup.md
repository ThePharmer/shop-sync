# Setup Instructions

## Prerequisites
- Python 3.9+
- PostgreSQL
- Etsy API credentials

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r backend/requirements.txt`
3. Copy `config.py.example` to `config.py` and add your credentials
4. Run database migrations (coming soon)
5. Start the development server (coming soon)
